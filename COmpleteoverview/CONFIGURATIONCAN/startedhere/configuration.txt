Configuration Strategy:
-----------------------

AUTOSAR CAN Stack Layers (Bottom-Up):
--------------------------------------
1. CAN Driver (MCAL Layer)
   - Hardware abstraction
   - Mailbox configuration
   - Interrupt handling
   
2. CanIf (CAN Interface) ← STARTING HERE
   - Software filtering
   - L-PDU handling
   - HOH (Hardware Object Handle) mapping
   
3. CanTp (CAN Transport Protocol)
   - Segmentation/Reassembly
   - Flow control
   - N-PDU handling
   
4. PduR (PDU Router)
   - Routing tables
   - Gateway functionality
   
5. COM (Communication)
   - Signal packing/unpacking
   - I-PDU handling
   
6. ComM (Communication Manager)
   - Network state management
